# Imágenes Necesarias

Este directorio debe contener las siguientes imágenes:

## hero.jpg
- **Tamaño recomendado:** 1920x1080px o similar
- **Uso:** Imagen de fondo para la sección Hero en la página principal
- **Formato:** JPG optimizado
- **Sugerencia:** Imagen de mascotas (perros, gatos) con buena iluminación

## logo.png
- **Tamaño recomendado:** 512x512px (cuadrado)
- **Uso:** Logo de VETSA Clinic (actualmente se usa un placeholder con la letra "V")
- **Formato:** PNG con fondo transparente

Para agregar estas imágenes, simplemente colócalas en este directorio con los nombres exactos especificados.
